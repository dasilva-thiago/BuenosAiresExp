# Buenos Aires Explorer

A personal project, desktop application built with C# and Windows Forms for managing
points of interest in Buenos Aires, supporting location registration,
distance calculation, and visit route planning.

> This project is under active development. Features and structure may change.

## Tech Stack

- C# / .NET 10
- Windows Forms
- Entity Framework Core 10.0.5
- SQLite
